Minilife v0.1.0
 ================= by nick
 
Uses Gosu (libgosu.org) and Ruby (ruby-lang.org)

How to use

    Mouse: Move the mouse to the borders of the screen to move the view in that direction
    Left click: Toggle a cell.
    Scroll up/down: Change zoom level.
    Enter: Run the pattern.
    Spacebar: Advance the pattern one generation.
    N: Erase all cells.
    Escape: Quit Minilife.
